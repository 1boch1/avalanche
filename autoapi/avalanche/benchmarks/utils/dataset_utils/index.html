class avalanche.benchmarks.utils.dataset_utils.TensorDatasetWrapper(tensor_dataset: ITensorDataset[T_co])[source]¶

Bases: DatasetWithTargets[T_co]

A Dataset that wraps a Tensor Dataset to provide the targets field.

A Tensor Dataset is any dataset with a “tensors” field. The tensors field must be a sequence of Tensor. To provide a valid targets field, the “tensors” field must contain at least 2 tensors. The second tensor must contain elements that can be converted to int.

Beware that the second element obtained from the wrapped dataset using __getitem__ will always be converted to int, This differs from the behaviour of PyTorch TensorDataset. This is required to keep a better compatibility with torchvision datasets.

Creates a TensorDatasetWrapper instance.

Parameters

tensor_dataset – An instance of a TensorDataset. See class description for more details.

avalanche.benchmarks.utils.dataset_utils.manage_advanced_indexing(idx, single_element_getter, max_length)[source]¶

Utility function used to manage the advanced indexing and slicing.

If more than a pattern is selected, the X and Y values will be merged in two separate torch Tensor objects using the stack operation.

Parameters

  • idx – Either an in, a slice object or a list (including ndarrays and torch Tensors) of indexes.

  • single_element_getter – A callable used to obtain a single element given its int index.

  • max_length – The maximum sequence length.

Returns

A tuple consisting of two tensors containing the X and Y values of the patterns addressed by the idx parameter.

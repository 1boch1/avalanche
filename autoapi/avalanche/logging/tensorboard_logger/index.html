log_metric(self, metric_value: MetricValue, callback: str)[source]¶

Helper method that will be invoked each time a metric value will become available. To know from which callback the value originated, the callback parameter can be used.

Implementing this method is a practical, non-exclusive, alternative the implementation of the single callbacks. See the class description for details and hints.

Parameters

  • metric_value – The value to be logged.

  • callback – The callback (event) from which the metric value was obtained.

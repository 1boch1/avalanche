avalanche.evaluation.metric_utils.default_cm_image_creator(confusion_matrix_tensor: Tensor, display_labels=None, include_values=True, xticks_rotation='horizontal', values_format=None, cmap='viridis', dpi=100, image_title='')[source]¶

The default Confusion Matrix image creator. This utility uses Scikit-learn ConfusionMatrixDisplay to create the matplotlib figure. The figure is then converted to a PIL Image.

For more info about the accepted graphic parameters, see: https://scikit-learn.org/stable/modules/generated/sklearn.metrics.plot_confusion_matrix.html#sklearn.metrics.plot_confusion_matrix.

Parameters

  • confusion_matrix_tensor – The tensor describing the confusion matrix. This can be easily obtained through Scikit-learn confusion_matrix utility.

  • display_labels – Target names used for plotting. By default, labels will be used if it is defined, otherwise the values will be inferred by the matrix tensor.

  • include_values – Includes values in confusion matrix. Defaults to True.

  • xticks_rotation – Rotation of xtick labels. Valid values are ‘vertical’, ‘horizontal’ or a float point value. Defaults to ‘horizontal’.

  • values_format – Format specification for values in confusion matrix. Defaults to None, which means that the format specification is ‘d’ or ‘.2g’, whichever is shorter.

  • cmap – Must be a str or a Colormap recognized by matplotlib. Defaults to ‘viridis’.

  • dpi – The dpi to use to save the image.

  • image_title – The title of the image. Defaults to an empty string.

Returns

The Confusion Matrix as a PIL Image.

avalanche.evaluation.metric_utils.get_task_label(strategy: PluggableStrategy) → int[source]¶

Returns the current task label.

The current task label depends on the phase. During the training phase, the task label is the one defined in the “train_task_label” field. On the contrary, during the test phase the task label is the one defined in the “test_task_label” field.

Parameters

strategy – The strategy instance to get the task label from.

Returns

The current train or test task label.

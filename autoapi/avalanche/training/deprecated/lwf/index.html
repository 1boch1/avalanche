Learning without Forgetting Strategy.

paper: https://arxiv.org/abs/1606.09282 original implementation (Matlab): https://github.com/lizhitwo/LearningWithoutForgetting reference implementation (pytorch): https://github.com/arunmallya/packnet/blob/master/src/lwf.py

Parameters

  • model – pytorch basic model.

  • classes_per_task – number of classes for each task.

  • alpha – distillation loss coefficient.

  • distillation_loss_T – distillation loss temperature.

  • warmup_epochs – number of warmup epochs training only the new parameters.

  • optimizer – pytorch optimizer.

  • criterion – pytorch optimization criterion.

  • mb_size – mini-batch size for SGD.

  • train_ep – training epochs for each task/batch.

  • device – device on which to run the script.

  • preproc – preprocessing function.

  • eval_protocol – avalanche evaluation protocol.

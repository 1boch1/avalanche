JointTraining

JointStrategy is a super class for all the joint training strategies.
class avalanche.training.strategies.joint_training.JointTraining(model: Module, optimizer: Optimizer, criterion, evaluation_protocol: Optional[EvalProtocol] = None, train_mb_size: int = 1, train_epochs: int = 1, test_mb_size: int = 1, device='cpu', plugins: Optional[Sequence[StrategyPlugin]] = None)[source]¶

Bases: avalanche.training.strategies.BaseStrategy

JointStrategy is a super class for all the joint training strategies. This means that it is not a continual learning strategy but it can be used as an “offline” upper bound for them. This strategy takes in input an entire stream and learn from it one shot. It supports unique tasks (i.e. streams with a unique task label) or multiple tasks.

Parameters

  • model – PyTorch model.

  • optimizer – PyTorch optimizer.

  • criterion – loss function.

  • evaluation_protocol – evaluation plugin.

  • train_mb_size – mini-batch size for training.

  • train_epochs – number of training epochs.

  • test_mb_size – mini-batch size for test.

  • device – PyTorch device to run the model.

  • plugins – (optional) list of StrategyPlugins.

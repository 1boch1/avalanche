BaseStrategy

BaseStrategy is the super class of all continual learning strategies.
class avalanche.training.base_strategy.BaseStrategy(model: Module, optimizer: Optimizer, criterion, evaluation_protocol: Optional[EvalProtocol] = None, train_mb_size: int = 1, train_epochs: int = 1, test_mb_size: int = 1, device='cpu', plugins: Optional[Sequence[StrategyPlugin]] = None)[source]¶

BaseStrategy is the super class of all continual learning strategies. It implements a basic training loop and callback system that can be customized by child strategies. Additionally, it supports plugins, a mechanism to augment existing strategies with additional behavior (e.g. a memory buffer for replay).

Parameters

  • model – PyTorch model.

  • optimizer – PyTorch optimizer.

  • criterion – loss function.

  • evaluation_protocol – evaluation plugin.

  • train_mb_size – mini-batch size for training.

  • train_epochs – number of training epochs.

  • test_mb_size – mini-batch size for test.

  • device – PyTorch device to run the model.

  • plugins – (optional) list of StrategyPlugins.

avalanche.benchmarks.datasets_envs.cifar.cifar.read_data_from_png(params)[source]¶

Extract the images and pickle them. Values are rescaled from [0, 255] down to [-0.5, 0.5]. If the number of imgs is too high this function flush it in batches. It returns the final lists of pixels, qualities and labels.

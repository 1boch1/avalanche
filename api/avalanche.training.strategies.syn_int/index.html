Synaptic Intelligence Strategy.

This is the Synaptic Intelligence pytorch implementation of the algorithm described in the paper “Continual Learning Through Synaptic Intelligence” (https://arxiv.org/abs/1703.04200)

Parameters

  • model – pytorch basic model.

  • optimizer – pytorch optimizer.

  • criterion – pytorch optimization criterion.

  • mb_size (int) – mini-batch size for SGD.

  • train_ep (int) – training epochs for each task/batch

  • multi_head (bool) – multi-head or not

  • device (device) – device on which to run the script.

  • preproc – prepocessing function.

  • eval_protocol – avalanche evaluation protocol.

  • lr – learning rate for the optimizer.

  • momentum – momentum used

  • l2 – weights decay regularization.

  • si_lambda – Synaptic Intellgence lambda term.

avalanche.benchmarks.new_cdata_loaders.core50 module¶

This module contains the high-level CORe50 scenario generator. It basically returns a iterable scenario object GenericCLScenario given a number of configuration parameters.

avalanche.benchmarks.new_cdata_loaders.core50.CORe50(root=None, scenario='nicv2_391', run=0)[source]¶

CORe50 continual scenario generator

Parameters

  • root – Path indicating where to store the dataset and related metadata. By default they will be stored in avalanche/datasets_envs/core50/data/.

  • scenario – CORe50 main scanario. I can be chosen between ‘ni’, ‘nc’, ‘nic’, ‘nicv2_79’, ‘nicv2_196’ or ‘nicv2_391.’

  • run – number of run for the scenario. Batch ordering change based on this parameter (a number between 0 and 9).

Returns

it returns a GenericCLScenario instance that can be

iterated.
